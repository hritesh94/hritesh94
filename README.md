### 👨‍💻 About Me

Aspiring roboticist building affordable, open-source robotics solutions. My long-term goal is to make robotic prosthetics accessible for differently-abled people worldwide.

- 🤖 Built a **5-DOF robotic arm** with gripper — 3D printed, Arduino-driven
- 🔬 Independently reproduced **GRACE & Key-Value Bottleneck** continual learning experiments and proposed a novel double-layer meta-key architecture
- 💡 Engineered a **laser-based wireless communication system** (Li-Fi prototype)
- 📡 Researched **OFDM/PAPR reduction** for wireless communication
- 🎯 Currently learning: **ROS 2** · **Gazebo** · **SLAM**
